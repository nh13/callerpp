Consensus Caller Plus Plus
====

A simple consensus caller based on [spoa](https://github.com/rvaser/spoa).

## Compilation

Install [spoa](https://github.com/rvaser/spoa).

Type `make`.

## Usage

Run `./bin/callerpp -h` to see all the options.
